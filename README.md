# martenw.com

Personal website, built using [quarto](quarto.org) and a customised version of my [pressmark theme](https://mdwm.org/quarto-pressmark/). Inspired by the [Financial Times](ft.com) and [Tufte CSS](https://edwardtufte.github.io/tufte-css/). 

**Check it out [→ here](https://martenw.com).**

If you want to recreate it, I suggest taking a closer look at the _pressmark_ theme. It's the basis for this site and allows you to create a similar one. _Please don't copy it exactly; be creative! It's your website, make it original._
